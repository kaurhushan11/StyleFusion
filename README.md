# MERN E-Commerce Website
As this is E-commerce fashion website so, there is an option of add to cart. There is an admin panel with which we can add and delete the products.

## Installation:
1. Backend
   ```
   cd backend
   npm install
   ```
2. Frontend
   ```
   cd frontend
   npm install
   ```
3. Admin
   ```
   cd admin
   npm create vite@latest .
   //(i) Select react
   //(ii) Select Javascript
   npm install
   ```

## Run the project
1. Backend
   ```
   cd backend
   node index.js
   ```
2. Frontend
   ```
   cd frontend
   npm start
   ```
3. Admin
   ```
   cd admin
   npm run dev
   ```
